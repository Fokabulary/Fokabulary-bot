# WebApp setup
## Client side

### Installation and setup
#### Install node
- Install node from [nodejs.org](https://nodejs.org/en/download/).

#### Install Dependencies
```
npm install
```

### Development
```
npm run dev
```

### Build
```
npm run build
```
 
## Server side

### Dependencies
```
pip install -r requirements.txt
``` 
### Develeopment
```
python app.py
```
